HTML Slidy

  • Dave Raggett’s Slidy is a tool to create slideshows in HTML / XHTML
  • it relies on a simple micro-format and a JavaScript implementation
  • most of Slidy’s features can be defined (and extended!) with a simple timesheet

Outline Lists With Hidden Content

You can make your bullet points or numbered list items into outlines that you can expand or collapse:

  • Add class="outline" to the <ul> or <ol> element. (Click on this list item for more details)
    • Slidy will then treat the list as an outline list.
    • Clicking on outline list items will expand/collapse block-level elements within that list item.
  • Add class="expand" to any <li> elements that you want to start in an expanded state.
    • All block-level elements within the outline list items are hidden unless class="expand" is specified.
    • Such pre-expanded items can be collapsed by clicking on them (not supported on IE6).

Accordion Lists

Here’s a variant of the previous slide. This is not supported by Slidy.

  • Add class="accordion" to the <ul> or <ol> element. (Click on this list item for more details)
    • Slidy will then treat the list as an accordion list.
    • Clicking on outline list items will expand/collapse block-level elements within that list item.
  • Compared to the outline class, there are two differences. (Click on this list item for more details)
    • Only one list item can be expanded at a time.
    • class="expand" is not supported (yet).
